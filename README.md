### Other npm scripts

1. `npm run dev`: run app in local
2. `npm run build`: build the app for production
3. `npm start`: run app in production
4. `npm run type-check`: typescript syntax checking
5. `npm run lint`: run eslint for all changed files
6. `npm run format`: auto format all changed files
